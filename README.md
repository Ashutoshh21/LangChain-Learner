# LangChain Learning Repository

A structured daily repository tracking my progress learning LangChain architecture, production AI patterns, and LLM orchestration.

## 🛠️ Tech Stack & Integrations Used
- **Orchestration:** LangChain (Modern partner package architectures)
- **Cloud LLMs:** Groq Cloud API (`openai/gpt-oss-20b`)
- **Open Source LLMs:** Hugging Face Serverless Inference API (`meta-llama/Meta-Llama-3-8B-Instruct`)
- **Local Embeddings:** Hugging Face `all-MiniLM-L6-v2` via `sentence-transformers`

## 📁 Repository Strategy
To save local storage space on my Mac Air M2, this repository leverages high-speed free cloud endpoints (Groq/Hugging Face) for heavy generation models while maintaining localized architectures for text embedding matrices.
